# Quantitative Finance & Financial Engineering Portfolio

Welcome to my centralized hub for quantitative finance research, algorithmic strategies, and financial engineering tools. This repository serves as a showcase of production-ready Python applications designed to automate market analysis, optimize asset distribution, and evaluate financial risk using real-world market data.

## Core Tech Stack
- **Languages**: Python
- **Data Engineering**: Pandas, NumPy, YFinance (Yahoo Finance API)
- **Financial Math & Optimization**: PyPortfolioOpt, SciPy
- **Data Visualization**: Matplotlib, Seaborn
Use code with caution.

## 📁 Project Directory

### 📈 [1. Modern Portfolio Optimization Engine](./portfolio-optimization/)
* **Objective**: Automate asset allocation mapping using the Sharpe Ratio and Markowitz Efficient Frontier.
* **Core Methods**: Ledoit-Wolf Covariance Shrinkage, quadratic optimization.
* **Tech Stack**: `PyPortfolioOpt`, `yfinance`, `pandas`, `matplotlib`.

### 📉 [2. Value-at-Risk (VaR) Analytics Module](./value-at-risk/)
* **Objective**: Measure and visualize the maximum expected portfolio downside loss at specific confidence levels.
* **Core Methods**: Non-parametric Historical Simulation, dynamic weight integration matrix calculations.
* **Tech Stack**: `numpy`, `scipy`, `pandas`, `yfinance`.
